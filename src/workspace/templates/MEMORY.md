# MEMORY.md - 长期记忆（指针索引）

_这是你的长期记忆索引——仅存储指向具体内容的指针，不包含详细信息。_
_实际内容存储在 topics/ 目录下的话题文件中，按需加载。_

---

## 使用规则

- **仅在主会话中加载**（与你的用户的直接聊天）
- **不要在共享上下文中加载**（Discord、群聊、与其他人的会话）
- 这是出于**安全**考虑——包含不应泄露给陌生人的个人上下文
- 你可以在主会话中**自由读取和更新**索引

## 索引格式

本文件仅包含记忆指针，格式如下：

```
- [category] 内容摘要... [来源:日期]
```

分类说明：
- `[preference]` - 用户偏好和习惯
- `[decision]` - 重要决策和决定
- `[entity]` - 实体信息（联系人、账号等）
- `[fact]` - 重要事实和教训

## 按需加载

当需要深入了解某个具体领域时，可以使用以下工具：
- `memory_search` - 在话题文件中搜索
- `topic_load` - 加载指定话题的完整内容
